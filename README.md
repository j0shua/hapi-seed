# Hapi Seed


#### to run the app:
- `npm install`
- `NODE_ENV=local PORT=9999 ./node_modules/rejoice/bin/rejoice -c config/manifest.js -p .`
or
- `NODE_ENV=local npm run start`

> if you are using vscode you can run the app + debgger by pressing `F5` 

#### to run the tests:
- `npm test` or `gulp test`
